# 第8章 辅助存储器

> 辅助存储器（外存）容量大、成本低、非易失，用于长期保存程序和数据，包括磁盘、磁带、光盘和固态盘等。

## 8.1 种类与技术指标

### 六大技术指标

| 指标 | 说明 | 典型值 |
|------|------|--------|
| **存储密度** | 道密度（tpi）+ 位密度（bpi）或面密度 | 磁道密度数百~数千tpi |
| **存储容量** | 格式化容量（用户可用）≈ 非格式化容量×60%~70% | — |
| **寻址时间** | 平均寻址时间 = 平均找道时间 + 平均等待时间 | 磁盘~10ms |
| **数据传输率** | D_r = 记录密度 × 介质运动速度 | — |
| **误码率** | 出错位数 / 读出总位数 | 10^-8~10^-12 |
| **价格** | 位价格 = 设备价格 / 容量 | 磁盘 < 内存 < Cache |

## 8.2 磁记录原理与记录方式

### 读写原理

- **写入**：磁头线圈通电流 → 磁头磁化 → 工作间隙产生漏磁场 → 介质表面磁性粒子磁化形成磁化单元
- **读出**：介质运动切割磁力线 → 读出线圈产生感应电压 e ∝ -N·dΦ/dt

### 磁记录方式

| 方式 | 特点 | 编码效率 | 自同步 |
|------|------|:--------:|:------:|
| **RZ（归零制）** | 正脉冲=1，负脉冲=0，位间电流为0 | 低 | 无 |
| **NRZ（不归零制）** | 始终有电流，连续1或0时不改变方向 | 100% | 无 |
| **NRZ1（见1就翻）** | 遇1翻转方向，遇0不变 | 100% | 无 |
| **PM（调相制）** | 利用相位差180°代表0和1 | 50% | 有 |
| **FM（调频制）** | 1在中心翻转且位间翻转，0仅位间翻转 | 50% | 有 |
| **MFM（改进调频制）** | 1在中心翻转，连续0时在起始翻转 | 100% | 有 |

**编码效率** η = 位密度 / 最大磁化翻转密度

**PRML（部分响应最大似然）技术**：利用符号间干扰的规律性，通过计算观测消除干扰，进一步提高记录密度。

### 磁头技术

- **感应式磁头**：环形导磁体绕线圈，有工作间隙
- **MR磁头（磁致电阻效应）**：专用读出磁头，输出灵敏度高且与转速无关
- **浮动式磁头**：硬盘磁头利用气垫悬浮于盘片表面，浮动间隙越小密度越高

## 8.3 磁盘存储器

### 温彻斯特磁盘

1973年IBM 3340首先采用。密封组合式，磁头、盘片、电机等制成不可拆卸整体，防尘好、可靠性高。

### HDD组成

1. **磁头定位驱动系统**：步进电机（开环，精度低）或音圈电机（闭环，速度快精度高）
2. **主轴系统**：转速3600~15000rpm
3. **数据控制系统**：读写电路、磁头选择、索引区标等
4. **盘组**：记录面→磁道（同心圆）→扇区

磁盘地址格式：[记录面号] [磁道号] [扇区号]

### 磁盘Cache

由SRAM或DRAM组成（十余MB以上），Cache行是基本单位，采用"写通"或"写回"策略。预读策略利用局部性规则对顺序读操作特别有效。

### RAID（独立冗余磁盘阵列）

1988年D.A. Patterson提出，基础是数据分块技术。

| 级别 | 名称 | 特点 | 容错 |
|:----:|------|------|:----:|
| 0 | 无冗余数据分块 | I/O性能最高，无容错 | 无 |
| 1 | 镜像磁盘阵列 | 工作盘+镜像盘 | 盘故障可恢复 |
| 2 | 海明码+位交叉 | 可单纠错双检错 | 盘故障可恢复 |
| 3 | 奇偶校验+位交叉 | 一个奇偶校验盘，适合大批量数据 | 一个盘故障 |
| 4 | 奇偶校验+块交叉 | 独立访问 | 一个盘故障 |
| 5 | 奇偶校验+块交叉分布 | 校验信息分布到所有盘，无专用校验盘 | 一个盘故障 |
| 6 | 两种奇偶校验+块交叉 | 可靠性最高 | 两个盘故障 |
| 10 | RAID 0+RAID 1 | 分块+镜像，性能和安全性兼备 | 盘故障可恢复 |

## 8.4 磁带存储器

- **顺序存取设备**，存取时间比磁盘长，但容量大、位价格低、格式统一
- 数据流磁带机：数据块间不启停，简化结构
- **记录方式**：线性记录（磁道与带边平行）和螺旋扫描记录（磁道呈斜线，道密度高）
- LTO-5：容量1.5TB/3.0TB（压缩），传输速度140~280MB/s
- CRC纠错：CRC + 横向奇校验 + 纵向偶校验，可定位并纠正单磁道错误

## 8.5 光盘存储器

激光聚焦成直径＜1μm的小光点进行读写。

### 类型

| 类型 | 容量 | 特点 |
|------|------|------|
| CD-ROM | 650MB | 只读，误码率＜10^-12 |
| CD-R（WORM） | 650MB | 用户写入一次，不可修改 |
| DVD | 4.7~17GB | 单/双面单/双层，红色激光650nm |
| 蓝光光盘 | 25~100GB | 蓝色激光405nm |

### 读写原理

- **形变**：激光热作用在薄膜上形成凹坑（=1），无坑=0
- **相变**：晶态与非晶态的反射特性不同
- **磁光（M-O）**：热磁效应，检测光的偏振方向

## 8.6 固态盘（SSD）

采用半导体存储介质（NAND Flash或DRAM）和传统磁盘接口，无机械旋转活动。

- 随机寻址速度比HDD快3个数量级
- 传输速度恒定，不受内外圈影响
- 防震，低功耗，无噪音
- 成本高于HDD，寿命受限于Flash写入次数
