# 计算机组成原理 · 知识地图

> 计算机组成原理研究计算机的内部结构、功能部件及其工作原理，介于软件和硬件之间的交界面上。

---

## 课程知识体系

```
高级语言程序（软件）
       ↓ 编译/解释
   指令集体系结构（ISA）      ← 第5章
       ↓ 译码/执行
   微体系结构（CPU）          ← 第6章
       ↓ 数据通路
   运算器 + 存储器 + I/O     ← 第3、4、7、8、10章
       ↓
   数字逻辑电路（硬件）       ← 第2章
```

### 各章节关系

| 章节 | 核心内容 | 前置知识 |
|------|---------|---------|
| 第1章 计算机系统概述 | 冯·诺依曼结构、层次结构、性能指标 | 无 |
| 第2章 计算机的逻辑部件 | 组合逻辑、时序逻辑、阵列逻辑 | 第1章 |
| 第3章 运算方法和运算部件 | 数据表示、四则运算、浮点运算、校验码 | 第2章 |
| 第4章 主存储器 | SRAM/DRAM、ROM、多体交叉 | 第1章 |
| 第5章 指令系统 | 指令格式、寻址方式、CISC/RISC | 第1、3章 |
| 第6章 中央处理器 | 数据通路、微程序、硬布线、流水线 | 第3、5章 |
| 第7章 存储系统 | Cache、虚拟存储器、MMU | 第4章 |
| 第8章 辅助存储器 | 磁盘、光盘、SSD | 第4章 |
| 第9章 输入输出设备 | 显示器、打印机、汉字处理 | 第1章 |
| 第10章 输入输出系统 | 中断、DMA、通道、总线接口 | 第5、7章 |

## 学习路线

1. **第1章** → 建立计算机整体层次结构认知
2. **第2章** → 数字逻辑电路基础（组合+时序+PLD）
3. **第3章** → 数据表示和运算方法（计算核心，分量最重）
4. **第4章** → 主存储器（SRAM/DRAM/ROM）
5. **第5章** → 指令系统（软硬件接口）
6. **第6章** → 中央处理器（核心部件，与第3、5章紧密关联）
7. **第7章** → 存储系统（Cache+虚拟存储器）
8. **第8章** → 辅助存储器（磁盘/光盘/SSD）
9. **第9章** → 输入输出设备（外设原理）
10. **第10章** → 输入输出系统（中断/DMA/总线）

> 建议优先攻破第3章和第6章，这两章计算题和设计题占比最高。
