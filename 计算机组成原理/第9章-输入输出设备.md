# 第9章 输入输出(I/O)设备

> 输入输出设备是计算机与外界交换信息的桥梁，包括输入设备（键盘、鼠标、触摸屏等）和输出设备（显示器、打印机等），以及汉字处理技术。

## 9.1 外部设备概述

CPU+主存储器构成主机，之外的硬件统称为外部设备。外设向多样化、智能化发展。

## 9.2 输入设备

### 9.2.1 键盘

按键开关排列成阵列，产生位置码→ASCII码或其他码。标准键盘101个键。

**ASCII码**：7位二进制表示128个字符。计算机内用8位（最高位为0或奇偶校验位）。
- 0~31：控制字符
- 32（20H）：空格
- 48~57（30H~39H）：数字0~9（高3位011，低4位0000~1001）
- 65~90（41H~5AH）：大写字母A~Z
- 97~122（61H~7AH）：小写字母a~z
- 127（7FH）：DEL

大写字母与小写字母仅b5一位不同。

**EBCDIC码**：IBM制定，8位，256个编码状态。

### 9.2.2 鼠标、跟踪球和操作杆

- **鼠标**：机械式（滚动球带动编码器）和光电式（光栅检测位移）。通信控制原理相同，输入相对坐标。
- **跟踪球**：与机械鼠标原理相同，球座固定。
- **操作杆**：便携机使用，应变规片或压敏电阻感受压力。

### 9.2.3 触摸屏

透明，安装在显示器屏幕外面。系统包含触摸屏控制器（卡）+触摸检测装置。

主要技术：电阻式、电容式、红外线式、表面声波技术。

### 9.2.4 图像输入设备

- 数码相机：CCD/CMOS传感器，JPEG压缩，内置存储器和LCD预览屏
- 摄像头/摄像机：图像数字化后存入闪存或磁盘

### 9.2.5 条形码

- **一维条码**：由宽度和反射率不同的"条"和"空"组成
- **二维条码**：从水平+垂直两个方向获取信息，容量更大
- 条码技术包括：编码规则、译码技术、印刷技术、光电扫描技术

### 9.2.6 OCR和语音输入

- **OCR系统**：扫描输入→光电变换→数字化→版面分析→字切分→识别
- 汉字识别难度远高于西文：字形多变、字体多、相似字多
- 字切分错误约占误识的30%

**模式识别四环节**：信息获取→预处理→特征提取→识别结果

## 9.3 输出设备——显示器

### 基本概念

| 术语 | 说明 |
|------|------|
| **分辨率** | 像素个数，越高越清晰（如1920×1080） |
| **灰度级** | 亮暗差别（8位=256级） |
| **刷新频率** | >30次/秒不闪，计算机选50帧/秒以上 |
| **帧存储器容量** | = 分辨率 × 灰度级位数 |
| **光栅扫描** | 逐行扫描或隔行扫描 |

### 显示器类型

| 类型 | 原理 | 特点 |
|------|------|------|
| **CRT** | 电子枪+偏转线圈+荧光屏 | 已被淘汰 |
| **LCD（液晶）** | 液晶分子偏转控制透光性，需背光源 | 低电压、微功耗、体轻薄 |
| **PDP（等离子）** | 惰性气体放电激发荧光粉 | 发光型，适合40in以上大屏幕 |
| **TFT-LCD** | 每个像素由薄膜晶体管驱动 | 主流技术 |
| **IPS-LCD** | 分子X/Y方向转动，视角达178° | 日立2001年提出 |
| **LED背光** | 替代CCFL，寿命10万小时 | 寿命远长于CCFL（2.5~6万小时） |

显示适配器（显卡）：核心为图形/图像加速芯片。接口标准：AGP → PCI Express。

## 9.4 输出设备——打印机

### 分类

| 分类方式 | 类型 |
|---------|------|
| **印字原理** | 击打式（针式）vs 非击打式（激光、喷墨、热转印）|
| **工作方式** | 串行（逐字）vs 行式（一次一行） |

### 点阵针式打印机

由n×m点阵组成字符（西文5×7~7×9，汉字16×16或24×24）。打印头装有单列多根针，印完一列后水平移动。可单向或双向打印。

彩色针式：四色带（黑蓝红黄），通过两次打印获得7种颜色。

仍在票据/多联打印等专用领域不可替代。

### 激光打印机

**六步印字过程**：带电（-600V）→ 曝光（激光→-100V潜像）→ 显影（墨粉附着）→ 转印（正电荷吸引）→ 定影（82℃熔化）→ 清除残像

分辨率达600dpi，速度为PPM（pages per minute，高速机100PPM以上）。

### 喷墨打印机

- **连续式**：压电陶瓷产生墨滴，充电偏转后回收无用墨滴
- **随机式**：仅在需印字时喷墨。压电式（60~200V脉冲）和热电式/气泡式

性能指标：喷嘴数、喷射频率。

### 发展趋势

击打式→非击打式。微型打印机和专用打印机（超市购物单、银行存折、机票等）不断发展。

## 9.5 汉字处理技术

### 汉字编码标准

- **GB 2312-1980**：6763个汉字（一级3755+二级3008）
- **ISO/IEC 10646（UCS）**：每个字符用4个8位表示。中日韩统一汉字在4E00~9FFF，共20902个位置

### 汉字输入方法

| 方法 | 说明 |
|------|------|
| **键盘输入** | 音码（全拼/简拼/双拼）、形码（500+部件组成6万+汉字）、音形码 |
| **语音输入** | 自然语言人机交互，难度大 |
| **手写输入** | 图形板+画笔+软件 |
| **印刷体扫描识别** | OCR仪 |

### 汉字存储与输出

- **字形码（字模码）**：点阵表示（16×16=32字节/字，24×24=72字节/字）
- 字模点阵存于字库中，显示时检索输出
- 字库发展：点阵→矢量→轮廓曲线
- 输出流程：键盘输入码→代码转换（查码表得机内码）→字形检索（查字模库得字形点阵）→输出
